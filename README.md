# My Setup

![](https://github.com/ALi3naTEd0/My-Setup/blob/main/screenshot.png)

### **OS**: [Arch](https://archlinux.org/)
### **Theme**: [Hyprdots](https://github.com/prasanthrangan/hyprdots)

## Install programs
```
paru -S alarm-clock-applet btop deemix-fix-gui-git discord element-desktop enpass-bin filelight firefox firefox-pwa fsearch git github-cli gnome-disk-utility gwenview hypnotix htop kdeconnect kid3 konsole linutil localsend mediainfo mkvtoolnix-gui nano net-tools nordvpn-bin notion-app-electron obsidian onlyoffice-bin plex-media-player plex-media-server-plexpass qbittorrent rclone rssguard rsync sox subtitleedit syncthing tauon-music-box telegram-desktop thunderbird uget webapp-manager -y
```

## Fonts
```
paru -S noto-fonts-cjk ttf-dejavu noto-fonts-emoji
```

## Fix VSCode Terminal
```
terminal.integrated.fontFamily": "'Cascadia Code', JetBrainsMono Nerd Font Propo, CaskaydiaCove Nerd Font Mono, Maple Mono"
```

## Install KVM/QEMU
```
paru -S qemu-full virt-manager virt-viewer dnsmasq bridge-utils libguestfs ebtables vde2 openbsd-netcat -y
```
